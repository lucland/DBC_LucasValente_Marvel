# dbc_lucas_valente

App Marvel para a DBC. Consumindo a Marvel API.
Para rodar o projeto, baixe o repositório na sua máquina e execute o comando "flutter pub get".
Gerencia de estado utilizada: Bloc (Cubit)
Modelo: MVC
